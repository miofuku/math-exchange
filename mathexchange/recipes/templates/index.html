What is a recipe?
A "mathematics recipe" is a collection of commands used to perform a mathematics data analysis task. A recipe can be thought of as a mathematics workflow or a mathematics pipeline.
Where are the recipes?
  • The Project List page contains recipes collected into projects.
  • The Latest Recipes page displays the most recently updated recipes.
What types of recipes can I find here?

Most recipes demonstrate typical use cases for various mathematics tools. Many recipes have been developed to support mathematics education.

Can I run recipes on my computer?

Yes! All recipes are designed to run both through this website and at the command line on your computer.
